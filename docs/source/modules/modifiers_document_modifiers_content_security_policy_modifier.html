Module modifiers/document/modifiers/content-security-policy-modifier

This file contains the source code for modifying the Content-Security-Policy <meta> tag in the document in order to put blocks on various resources in the document from loading or executing.

addAudioCSP

  • addAudioCSP(document: Document): void
    • Defined in modifiers/document/modifiers/content-security-policy-modifier.ts:256

    Adds a new Content-Security-Policy to block media from all sources (aliases the addVideoCSP function).

    command

    --add-audio-csp

    example
    // `<head>` tag before the Content Security Policy is added:
//
// <html>
//   <head></head>
// </html>

// Adding the Content Security Policy to block document media.
addAudioCSP(document);

// After running, the Content Security Policy is added:
//
// <html>
//   <head>
//     <meta http-equiv="Content-Security-Policy" content="media-src none">
//   </head>
// </html>

    Parameters

    • document: Document

      the document that will be modified in place.

    Returns void
