minifyFonts

  • minifyFonts(document: Document, fontCache: ResourceCache): void
    • Defined in modifiers/media/fonts/font-minifiers.ts:181

    Minifies all fonts in the provided font cache using a series of font minification and conversion algorithms. The algorithm used for conversions and minifications is as follows:

    • TTF -> WOFF2 -> Subsetted WOFF2 (using Fonttools)
    • OTF -> WOFF2 -> Subsetted WOFF2 (using Fonttools)
    • WOFF -> WOFF2 -> Subsetted WOFF2 (using Fonttools)
    • WOFF2 -> Subsetted WOFF2 (using Fonttools)
    command

    --minify-fonts

    dependencies

    Fonttools 4.

    Parameters

    • document: Document

      the document that contains text content used for the font subsetting process.

    • fontCache: ResourceCache

      a resource cache of fonts that will be minified.

    Returns void
