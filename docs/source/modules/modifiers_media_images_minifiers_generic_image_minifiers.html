minifyImageToJpeg

  • minifyImageToJpeg(bytes: Buffer, jpegQuality: number): Promise<Buffer>
    • Defined in modifiers/media/images/minifiers/generic-image-minifiers.ts:24

    Possibly converts an image buffer into a minified JPEG image buffer. If the image has transparency, no image conversion occurs and the original buffer is returned.

    Parameters

    • bytes: Buffer

      a buffer containing an image.

    • jpegQuality: number

      the quality of the JPEG that the image will be converted to.

    Returns Promise<Buffer>

    either the original image buffer or minified JPEG buffer.

    Parameters

      a buffer containing an image.

    • webpQuality: number
