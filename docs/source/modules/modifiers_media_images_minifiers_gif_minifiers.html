minifyGif

  • minifyGif(bytes: Buffer, jpegQuality?: number, allowWebp?: boolean, webpQuality?: number): Promise<Buffer>
    • Defined in modifiers/media/images/minifiers/gif-minifiers.ts:276

    Possibly minifies a GIF image buffer. This function runs through multiple algorithms to attempt to reduce the size of a GIF, including possible conversion of the GIF into different image formats so long as no features of the GIF are lost. If none of the minified images are smaller than the original image buffer, the original buffer is returned.

    dependencies
